# BEI QC分析ツール

builelib BEI計算エンジンの品質管理（QC）分析用のインタラクティブダッシュボードです。

## ファイル構成

| ファイル | 内容 |
|---|---|
| `qc_dashboard.html` | QC七つ道具ダッシュボード（メインページ） |
| `qc_dashboard.css` | ダッシュボードのスタイルシート |
| `qc_dashboard.js` | ダッシュボードのロジック・データ |
| `input_params_map.html` | 入力パラメータ全体マップ（157項目一覧） |

## 使い方

ブラウザで直接開けます（サーバー不要）：

```
# QC七つ道具ダッシュボード
qc_dashboard.html

# 入力パラメータ全体マップ  
input_params_map.html
```

## QC七つ道具

1. **パレート図** — 設備別エネルギー寄与度の可視化
2. **特性要因図** — BEI悪化要因の因果関係
3. **チェックシート** — パラメータ入力精査レベル判定
4. **ヒストグラム** — 基準一次エネルギー消費量の分布
5. **散布図** — 空調 vs 換気の相関分析
6. **管理図** — 地域別空調基準値の異常検知
7. **層別** — 建物用途別エネルギー構成比較
